Программа написана на языке C#. После запуска на экране отображается меню с несколькими пунктами. Пользователь вводит номер нужного пункта, после чего программа выполняет выбранное действие.

В программе используются:
вывод текста в консоль;
изменение цвета текста;
изменение положения курсора;
обработка ввода пользователя.

Запустите проект через Visual Studio, VS Code или командой dotnet run.
